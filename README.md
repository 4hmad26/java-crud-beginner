# Java CRUD Application

## Description
This is a basic console-based CRUD (Create, Read, Update, Delete) application developed in Java.
It is designed for beginners to understand core Java concepts and backend fundamentals.

## Features
- Create user
- Read user data
- Update user
- Delete user

## Technologies Used
- Java
- ArrayList
- Scanner

## How to Run
1. Compile the program:
   javac CRUDApplication.java
2. Run the program:
   java CRUDApplication
